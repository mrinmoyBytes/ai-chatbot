# 🤖 AI Chatbot using Groq LLaMA 3

A conversational AI chatbot built with Groq API and Streamlit.

## 🌐 Live Demo
[Click here to chat](https://ai-chatbot-vpma6eg8xmvbpboyuidzo2.streamlit.app/)

## Features
- 💬 Real AI responses using LLaMA 3
- 🧠 Chat history memory
- 🎨 Clean web interface
- ⚡ Super fast responses via Groq

## Tech Used
- Python
- Groq API (LLaMA 3)
- Streamlit

## Run Locally
pip install groq streamlit
streamlit run chatbot.py
